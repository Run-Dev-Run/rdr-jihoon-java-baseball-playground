## 구현 할 기능 목록 정리

- 컴퓨터는 3자리 난수를 생성한다.
  - 1~9까지의 수를 3자리 생성한다.
  - 각 숫자는 중복을 허용하지 않는다.
- 사용자는 3자리 난수를 입력한다.
  - 1~9까지의 수를 3자리 입력한다.
- 두 입력을 비교해 ?볼 ?스트라이크 여부를 판정한다.
  - 위치, 숫자 모두 같다면? -> strike
  - 위치가 다르고 숫자만 같다면? -> ball
  - 위치, 숫자 모두 같다면? -> nothing
- 결과를 출력한다.
- 게임 진행 모듈을 생성한다.
  - 게임 실행 모듈
  - 입력 모듈
  - 출력 모듈
